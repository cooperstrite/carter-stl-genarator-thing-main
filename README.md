# carter-stl-genarator-thing

Static GitHub Pages site for pasting code or uploading files (STL, OBJ, PLY, OFF, glTF/GLB, 3MF, SCAD), previewing them in 3D, and exporting a clean STL download. `fixer.html` provides a local \"AI fixer\" for SCAD/STL text. Open `index.html` locally or enable GitHub Pages for the repo root.
